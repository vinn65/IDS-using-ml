Intrusion Detection System Using ML

Predict

This is an illustration of how the intrusion detection system that uses a trained ensembler model to detect and predict Network intrutions. It uses three different Classifiers( Random Forest, Decision Tree, and Gaussian Naive Bayes) combined by a voting classifier to form an ensembler model hence increasing its accuracy. The model will also be able to detect new occuring attacks since it uses machine learning.
